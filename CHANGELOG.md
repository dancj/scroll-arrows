## [0.1.0] - 2026-06-19

Initial release.

### Added

- Framework-agnostic core that draws hand-drawn arrows between two elements,
  powered by [rough.js].
- Single `roughness` knob (0 → clean straight lines, 1 → scratchy curvy
  scribbles).
- Click-through overlay `<svg>` with endpoints auto-tracked via
  `ResizeObserver`, drawn on scroll progress.
- Obstacle routing and anchored ends.
- Thin React wrapper: `useScrollArrow` hook and `ScrollArrowLine` component.
- Dual ESM/CJS builds with bundled type declarations.
